Tyubox 待办盒子

一款纯净高效的待办应用，进度条、成就徽章、重复任务、数据备份……所有功能都在本地，干净利落。

✨ 核心功能 · 面面俱到

每个特性都经过深思熟虑，只为提供最流畅的待办体验。

📊
今日进度条

实时显示今日完成率，每完成一项任务进度条平滑增长。全部完成时解锁「今日事今日毕」永久徽章。

🏆
徽章成就系统

累计完成10/50/100个任务、连续打卡3/7/30天等成就自动点亮，成就墙可随时查看。

🔄
智能重复任务

支持“每天跑步”“每周一开会”等自然语言，完成任务后自动生成下一个周期任务，截止日期智能推演。

🔔
浏览器通知

任务到期前一天或当天自动推送提醒，点击通知可直接定位到任务并高亮闪烁，不错过任何重要事项。

📥📤
数据备份与恢复

一键导出所有待办、已完成、归档数据为JSON文件，支持导入恢复，跨设备迁移无忧。

📈
统计图表 + 标签云

近7天完成趋势柱状图，标签云点击即可筛选任务，直观分析效率。

🔍
搜索与筛选联动

支持关键词搜索（文本/标签），点击标签云可快速筛选，筛选时搜索框自动提示当前范围。

📅
优先级 + 截止日期

高/中/低优先级视觉标识，截止日期逾期红色、今日到期金色高亮，任务一目了然。

🎨
主题色 & 深色模式

六种个性主题色（流光金/霓虹紫/赛博蓝...），一键切换深色/浅色模式，随系统或手动。

🗄️
任务归档

已完成任务可一键归档，归档列表支持恢复或永久删除，保持主列表清爽。

🎤
语音输入

支持浏览器语音识别，直接口述创建待办，解放双手。

📤
分享与导入代码

生成进度共享代码，朋友粘贴即可导入你的待办进度；也可导出文字列表分享。

⚙️ 技术实现 · 纯前端 隐私安全

核心架构
纯HTML/CSS/JS

无任何后端依赖，所有数据存储在浏览器本地（localStorage）。完全离线可用，不收集任何用户隐私。

自然语言解析
智能规则引擎

内置规则解析“明天”“紧急”“每天跑步”“每周一开会”等中文短语，自动设置截止日期、优先级、重复规则。

数据持久化
localStorage 存储

待办、已完成、归档均以JSON格式存储；支持完整导出/导入，数据跨设备迁移方便。

通知与交互
Web Notification API

到期提醒 + 点击跳转高亮，采用 requestAnimationFrame 实现平滑动画，体验流畅。

图表与徽章
Canvas 绘制 + 条件解锁

近7天图表动态绘制，徽章系统基于累计数据自动解锁，并持久化成就状态。

语音识别
Web Speech API

调用浏览器原生语音识别，无需额外权限（仅麦克风请求），快速创建任务。

🎯 设计哲学 · 极简 & 高效

我们不堆砌冗余功能，每个特性都经过深思熟虑：进度条激励、成就体系提供正向反馈；重复任务解放大脑；纯本地存储保护隐私。整个应用如一款精致的原生软件，开箱即用。
